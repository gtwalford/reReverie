By Gabriel Walford

An experiment using Canvas and WebGL through Processing.JS to create an explorable generative world within the browser.

Controls :

  • ' UP ' arrow - moves Forward
  • ' DOWN ' arrow - moves Backward
  • ' LEFT ' arrow - moves Left
  • ' RIGHT ' arrow - moves Right
  • ' r ' - moves Upward
  • ' f ' - moves Downward
  • ' w ' - angles Upward
  • ' s ' - angles Downward
  • ' a ' - angles Left
  • ' d ' - angles Right

** If the controls do not seem to be responding - click once on the window.
